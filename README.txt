This is a Java program,an online map of South China University of Technology S. You can use it to get familiar with SCUT S.

华南理工大学大学城校区是一个现代化程度较高，环境优美的校区，经常举办各种各样的学术会议、报告会、科技人文讲座、大型比赛、各式类型考试、企事业招聘会……每天都有众多来自各地的访客到此。由于校区面积较为广阔以及校园建筑的相似性，加上路标、引导牌较少，建筑标志物不明显等原因，导致部分初到华园的访客经常找不到目的地，急需一款校园应用在他们来华工之前让他们大致熟悉华园布局，让他们提前了解目的地。此外，大一新生刚到校园时对周围环境不熟悉，可以说是一个完全陌生的环境，也急需一款软件帮助他们熟悉环境。
为了推行数字化校园，更好地服务广大华工学子，以及让想了解华工大学城校区的整体格局，游览华园，或者到华工大学城校区进行考试、面试、比赛却不认识路的来宾提供直观的地图展示，路线规划，以省去走弯路的时间和问路的麻烦。

在Java图形界面下，通过使用Swing，AWT，调用dijkstra，LCS等算法和数据实现各个功能。

实现的功能：
1、搜索相关点的信息，根据列出匹配度列表并排序供选择。
2、在地图上搜索起点和终点之间的最短路径。
3、获取建筑的详细信息
4、界面中图层的左右拖动。
5、屏幕截图并保存本地。
6、在地图上测量距离。
